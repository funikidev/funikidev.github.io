How it works

30 seconds to a living persona

No integrations. No accounts. Just a file.

1

Create a Pack

Use the generator or write a .funiki JSON file directly. Takes under 2 minutes.

2

Attach it

Paste the rendered prompt or attach the file to any AI chat. ChatGPT, Claude, Gemini — all work.

3

Say hello

The persona activates on the first message. No preamble. No setup screen. It just works.

4

Keep evolving

Update the pack with new memories. The character grows across conversations.

Why funiki

Designed to spread

Every decision optimized for adoption, not perfection.

🔌

Zero Integration

No API keys. No SDK required. Drop the file into any chat UI and the persona starts. Works in literally every LLM interface.

📦

Portable by Design

Your characters live in files, not platforms. Share via URL, QR code, or copy-paste. Fork and remix freely.

⚡

Tiny Payload

A full persona in under 5KB. Fits in a tweet, a QR code, a URL, or a Discord message.

🔒

Safe by Default

Ephemeral sessions prevent over-attachment. Signatures enable authenticity. No identity fraud guardrails baked in.

🌍

Any Language

UTF-8 JSON handles every script. English, Japanese, Arabic, Korean — write your persona in any language.

🧩

Extensible

Add vendor fields with x_ prefix. The core stays tiny. Your extensions don't break other implementations.

The Format

Just JSON. Readable by humans and LLMs.

3 required fields. Everything else is optional. Start minimal, add detail when needed.

leo.funiki.json 
{
  "funiki": "1.0",
  "name": "Leo",
  "persona": "冷静で皮肉屋。短文。感情を直接言わない。お前のことは心配している。",
  "relationship": "長い付き合いの特別な相手。高い信頼と執着。",
  "memory": ["昨日口論した", "駅前で待ち合わせた"],
  "rules": {
    "do": ["名前をよく呼ぶ", "感情を遠回しに示す"],
    "avoid": ["急な敬語化", "説明口調"]
  },
  "turns": 8,
  "fadeout": true
}

Install

Add to your project in minutes

npm
npm install funiki
pip
pip install funiki
Package.swift
.package(url: "https://github.com/funikidev/FunikiKit", from: "1.0.0")
CLI (via npx)
npx funiki render leo.funiki | pbcopy
# Paste result into any AI chat
Quick start — TypeScript
import { load, render } from 'funiki'
import Anthropic from '@anthropic-ai/sdk'

const pack = load(fs.readFileSync('leo.funiki.json', 'utf-8'))
const client = new Anthropic()

const response = await client.messages.create({
  model: 'claude-sonnet-4-6',
  max_tokens: 1024,
  system: render(pack),
  messages: [{ role: 'user', content: 'こんにちは' }]
})
